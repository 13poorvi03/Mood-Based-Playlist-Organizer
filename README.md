
🎶 Mood-Based Playlist Organizer
A Python mini‑project that organizes songs into playlists based on mood. Using sets, dictionaries, lists, and loops, it helps you categorize tracks into moods like Happy, Sad, Chill, Study, or Workout. It also adds fun features like random mood shuffle, study booster, and mood suggestions.



✨ Features
Add songs to playlists by mood

Display all playlists neatly

Search songs by mood

Random mood shuffle (surprise pick)

Study booster (suggest instrumental tracks)

Mood suggestions (balance sad/happy vibes)




🛠 Tech Stack
Python (core logic)

Data structures: lists, sets, dictionaries, tuples

Loops for iteration and playlist management
